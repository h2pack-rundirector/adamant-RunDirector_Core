# Run Director Modpack

Run Director is a modular Hades II run-customization pack that gives you fine-grained control over run structure: which rooms appear, when special encounters can show up, which gods enter the pool, and which boons can be offered.

## What Run Director Provides

The gameplay modules focus on three parts of a run:

- Biome Control
  Shapes encounters inside each biome, including room timing for fountains, trials, shops, NPC rooms, minibosses, and route-specific reward behavior. It also includes controls for first-boon preferences and Dream Dive biome order.
- God Pool
  Controls the Olympian god pool before individual boon filtering happens. Choose which gods can appear, how many are allowed into the run, and whether special sources such as keepsakes can expand the pool.
- Boon Bans
  Filters individual boon offers across core Olympians, bonus gods, hammers, NPCs, and keepsake-style sources.

Together, these modules let you steer:

- which Olympian sources are allowed into the run
- which individual boons can appear once a source is active
- how biomes pace their special rooms, NPCs, minibosses, and reward structure

## How To Use

Install using r2modman. In game, open the Run Director menu and configure the modules from the shared settings window.
